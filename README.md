# angularjs-components
General AngularJS components that I created in my projects.

## Directives:
### money input directive
Usage: 
    
    <ed-money-input 
        name="numOfHours"
        blurhandler="checkTimes()"
        focushandler=""
        model="day.numOfHours"
        disabled="false">
    </ed-money-input>
    
### time input directive

        disabled="false">
    </ed-time-input>
